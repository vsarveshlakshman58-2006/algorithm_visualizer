# Sorting Algorithm Visualizer

A Python-based visualization tool that demonstrates how sorting algorithms work step-by-step.

## Algorithms Implemented

- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

## Technologies Used

- Python
- Matplotlib
- NumPy

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the program

python main.py
